# Student Grade Manager CLI

A simple command-line application built with Python to manage student records and practice core Python concepts.


## Features

- Add a student and their score
- View all students
- Search for a student by name
- Show the highest score
- Calculate the average score
- Delete a student
- Sort students by score


## What I Learned

While building this project, I practiced:

- Functions
- Lists
- Tuples
- Loops (`while`, `for`)
- Conditional statements (`if`, `elif`, `else`, `match-case`)
- Exception handling (`try` / `except`)
- User input validation
- Searching and deleting items from a list
- Sorting using the `key` parameter
- Basic Git and GitHub workflow
